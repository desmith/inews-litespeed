WP Page Post Widget Clone plugin creates a clone of a page or post and widgets sidebar with the same content as the original with it's contents and settings in just one click.

== Description ==
With one click, this plugin can create a replica or clone of a post or page. Now you don't need to do the styling of a new page or post from scratch.

Major features of this plugin include:

*   Create a page duplicate preserving all the content and title and with all settings.
*   Create a post duplicate preserving all the content and title and with all settings.
*   Create a widget sidebar duplicate preserving all the content and title and with all settings.
*   Cross Browsers Support ( Firefox, Chrome, Safari, Opera, etc. )


How to Use and Install

	* Download from wordpress.org or direct install and activate the plugin.
	* To clone posts just go to all posts on your dashboard and then hover cursor over a post and click on "Clone Me" option. After that, you will be redirected to the draft copy, make the changes you wish and hit publish.
	* To clone pages just go to all pages on your dashboard and then hover cursor over a page and click on "Clone Me" option. After that, you will be redirected to the draft copy, make the changes you wish and hit publish.
	* To clone widget just go to Appearance->widgets on your dashboard and then top of the sidebar visible setting block from this select one copy from sidebar and select where you wish to clone it from paste in sidebar dropdown and hit on "Clone Widgets" button. After that, you will be visible duplicated sidebar.

== Installation ==

* Upload **wp-page-post-widget-clone.zip** file Via WordPress Admin > Plugins > Add New.
* Alternately upload the **wp-page-post-widget-clone** folder  to the `/wp-content/plugins/` directory via FTP.
* Activate the WP Page Post Widget Clone plugin through the 'Plugins' menu in WordPress dashboard.
* Go to all posts or all page and also go in widget for clone sidebar on your dashboard
* To clone pages just go to all pages on your dashboard and then hover cursor over a page and click on "Clone Me" option. After that, you will be redirected to the draft copy, make the changes you wish and hit publish.
* To clone posts just go to all posts on your dashboard and then hover cursor over a post and click on "Clone Me" option. After that, you will be redirected to the draft copy, make the changes you wish and hit publish.
* To clone widget just go to Appearance->widgets on your dashboard and then top of the sidebar visible setting block from this select one copy from sidebar and select where you wish to clone it from paste in sidebar dropdown and hit on "Clone Widgets" button. After that, you will be visible duplicated sidebar.
